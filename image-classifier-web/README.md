# Image Classifier Web (Scaffold)

This is a minimal scaffold for an image-classifier website.

Quick start

1. Create a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the server:

```bash
python server.py
```

4. Open http://localhost:5000 in your browser.

Next steps

- Replace the placeholder prediction in `server.py` with your model loading and inference code.
- Add model files and update `/predict` to return real predictions.
